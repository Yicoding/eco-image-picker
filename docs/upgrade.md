# 从 v1 到 v2

### 去除 antd-mobile 组件库依赖

- `v2` 版本将不再依赖 `antd-mobile`，更纯净的第三方图片/文件上传组件。

### 组件拆分

- ImagePicker
  - `ImagePicker` 拆分为 `ImagePicker` 和 `FilePicker` 两个组件
- ImageView
  - `ImageView` 拆分为 `ImageView` 和 `FileView` 两个组件
