## API

### Toast.show(options)

| 属性     | 说明           | 类型      | 默认值  |
| :------- | :------------- | :-------- | :------ |
| text     | 提示内容       | `string`  | -       |
| duration | 时长           | `number`  | `2500`  |
| mask     | 是否显示遮罩层 | `boolean` | `false` |
