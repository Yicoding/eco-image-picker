## API

| 属性     | 说明           | 类型              | 默认值  |
| :------- | :------------- | :---------------- | :------ |
| visible  | 是否显示 modal | `boolean`         | `false` |
| children | 子元素         | `React.ReactNode` | -       |
| footer   | 底部按钮       | `Array`           | `[]`    |
| onClose  | 关闭回调       | `() => void`      | -       |
