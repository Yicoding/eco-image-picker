# eco-image-picker

基于`react`的移动端图片上传组件

[查看文档和示例][site]

## 使用

### npm 或 yarn 安装

```shell
npm install eco-image-picker --save
```

<br />

```shell
yarn add eco-image-picker
```

<br />

```shell
import ImagePicker from 'eco-image-picker';
```

[site]: https://yicoding.github.io/eco-image-picker
